# Data Visualization Mini Projects

## Overview
This repository contains two data visualization projects completed as part of 
coursework. Each notebook demonstrates the use of Python libraries to explore 
and visualize real-world datasets.

## Projects

### 1. Nightingale Rose Chart Recreation
A recreation of Florence Nightingale's famous polar area diagram (1858), 
which visualized causes of mortality among soldiers during the Crimean War. 
This project replicates the original chart using modern Python tools.

### 2. Telecom Customer Churn Analysis
A visual exploration of customer churn patterns in the telecommunications 
industry, identifying key factors associated with customer attrition.

## Repository Contents
| File | Description |
|------|-------------|
| `nightingale_mini_project_1.ipynb` | Florence Nightingale rose chart recreation |
| `teleco_project.ipynb` | Telecom customer churn visualization |

## Tools & Libraries
- Python
- Jupyter Notebook

2. Open any `.ipynb` file in Jupyter Notebook or JupyterLab
3. Run all cells in order
